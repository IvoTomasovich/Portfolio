Video Game Backlog Tracker

This program uses parsing to retrieve data from the video game website HowLongToBeat. Specifically, for any game link the user inputs, it fetches data on how long it takes to beat the main story, main + sides, completionist, and all styles. Once the user finishes, all this data is compiled into an Excel sheet for easy viewing. Additionally, there are columns that aggregate the numbers for each category, allowing users to see the total time needed to complete every style for all their games in the list. This program is designed to help users organize their entire gaming backlog conveniently. Furthermore, it provides average completion times for each game in various styles, reducing the need for users to repeatedly check HowLongToBeat for this information.

2021-2022 Midfielder Analysis

This project analyzes the performance and market value of top European soccer midfielders across major leagues (Premier League, La Liga, Serie A, Bundesliga, and Ligue 1). Using statistical modeling and data visualization techniques, the project identifies trends in passing efficiency, market values, and composite performance scores to determine which leagues offer the best value for midfielders. The analysis integrates datasets of player statistics and market values, leveraging Python libraries like Pandas, Matplotlib, and Seaborn for data processing, visualization, and insights. Key findings are presented through interactive charts and comparisons, showcasing the relationship between on-field performance metrics and market value. This project demonstrates strong analytical, data modeling, and visualization skills applied to a real-world sports dataset.
